## рҹҺ“ Key Formulas Implemented

### Expected Value (EV)
```
EV = (Win Rate Г— Average Win) - (Loss Rate Г— Average Loss)

Example:
Win Rate: 55%
Avg Win: $100
Loss Rate: 45%
Avg Loss: $50

EV = (0.55 Г— $100) - (0.45 Г— $50)
   = $55 - $22.50
   = $32.50 per trade
```

### Kelly Criterion
```
Kelly % = (p Г— r - q) / r

Where:
p = probability of win
q = probability of loss (1 - p)
r = reward/risk ratio

Example:
p = 0.60 (60% win rate)
q = 0.40 (40% loss rate)
r = 2.0 (2:1 reward:risk)

Kelly = (0.60 Г— 2.0 - 0.40) / 2.0
      = (1.20 - 0.40) / 2.0
      = 0.40 / 2.0
      = 0.20 (20% position size)
```

### Sharpe Ratio
```
Sharpe = (Average Return - Risk-Free Rate) / Standard Deviation

Example:
Avg Return: 2% per trade
Risk-Free Rate: 0%
Std Dev: 1.2%

Sharpe = (2% - 0%) / 1.2%
       = 1.67 (good risk-adjusted returns)
```
